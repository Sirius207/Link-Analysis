# Link-Analysis

Implementation and Analysis of HITS, PageRank, & SimRank

## Usage

### Calculate HITS, PageRank, & SimRank
```
python main.py --input data/graph_1.txt --output output.csv
```
- input: input data file name
- output: output file name
